# SEO Scope For Lab

## 1.1 Публичные страницы (индексируемые)
- `/` (перенаправление в публичный каталог)
- `/опросы`
- `/результаты`

## 1.2 Служебные/закрытые страницы (исключены из индекса)
- `/вход`
- `/голосование`
- `/успех`
- `/новый-опрос`
- `/профиль`
- `/админ`

## 1.3 Приоритетные страницы
- `/опросы` (главная точка входа в контент)
- `/результаты` (ключевая страница для выдачи по запросам про итоги голосований)
